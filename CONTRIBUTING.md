# Contributing to Maxis Linux

Thank you for wanting to contribute to Maxis Linux!

## How to contribute
1. Fork the repository
2. Create a new branch for your changes
3. Make your improvements or fixes
4. Submit a pull request

## Reporting issues
Please use the **Issues** tab for bug reports or suggestions.

## Notes
Maxis Linux is my first Linux distribution.  
Expect some experimental code and minor issues.
